# MS-DOS 

## Build from source

Mount the MSDOS folder in DOSBOX to run BUILD.BAT and compile the source code.

```
mount c .
build.bat
```

## Tools

Toolchain required: (masm61/bin)
```
  DOSXNT.EXE
  LIB.EXE
  LINK.EXE
  ML.ERR
  ML.EXE
```

## Precompiled binaries

You can find binary distributions in the bin folder.
